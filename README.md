# AC:NH Villagers
An application displaying all the villagers in the video game Animal Crossing: New Horizons

## Link to site
https://newhorizonsvillagers.herokuapp.com/villagers

## Technologies Used
* HTML
* CSS
* JavaScript
* EJS
* Node.js
* Express
* MongoDB
* Mongoose
* API from http://acnhapi.com/
